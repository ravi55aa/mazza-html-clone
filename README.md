# 🍹 Mazza Website Clone

A simple front-end clone of the Mazza beverage website built using **HTML** and **CSS**. This project focuses on recreating the visual design and layout of the original website while improving frontend development skills.

---

## 📌 Project Overview

This is a static website clone inspired by the official Mazza website. The goal of this project was to practice:

* Structuring web pages using HTML
* Styling and layout design using CSS
* Building visually appealing UI components
* Understanding responsive design basics

---

## 🚀 Features

* 🎨 Clean and modern UI inspired by Mazza
* 📱 Basic responsive layout
* 🧩 Structured sections (Hero, About, Products, etc.)
* 💡 Simple and readable code

---

## 🛠️ Tech Stack

* HTML5
* CSS3

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/ravi55aa/mazza-clone.git
   ```

2. Open the project folder

3. Run the project by opening `mazza.html` in your browser
